# VeSP

C++ implementation of the **Ve**rtex **S**earch by **P**rojecting method

---

Algorithm for finding the vertex of a multidimensional polytope of feasible solutions to a linear programming problem.

## Dependencies
- C++11 or higher
- [LinCore](https://github.com/zhulevae/LinCore) library

## Examples
- [3D cube with a clipped vertex](./examples/Clipped3DCube.cpp)
